    About

    This game is created as an exercise in using MySQL databases and Node.js.

    Resources

    • jQuery Libarary for easier DOM navigation and manipulatio
    • node.js as the backend framework
    • MySQL for the data storage
    • The music is from musicfox.com

    Game controls

    Press the arrow keys to move the tiles (number fields). All tiles then move to the side and adjacent tiles with the same value add up to a "new" tile.
